二、常用函数

2.1、ifnull(exp1,exp2)

如果第一个参数部位 null，则返回第一个参数，否则返回第二个参数，

ifnull(1/0,'louis')当exp1=1/0时，exp1为null，那么将返回第二个参数exp2

在mysql中null不能用=判断是否为null而是用is null

  • if(exp1,exp2,exp3)

    这个函数作用为：当exp1的值为true的时候(不为0或null就是true)，则返回exp2，否则返回exp3

2.2、round(x,d)

round用于数据的四舍五入，x指要处理的数，d是指保留几位小数，其中d也可以为负数，这时他表示小数点左边的d位数为0

三、窗口函数

定义：在满足某些条件的记录集合上执行的特殊函数，对于每条记录都要在此窗口内执行函数。右的函数随着记录的不同，窗口大小都是固定的，称为静态窗口；有的函数则相反，不同的记录对应着不同的窗口，称为滑动窗口。

窗口函数是mysql8.0之后才出现的功能。

3.1、聚合函数和窗口函数的区别

  • 聚合函数是将多条记录聚合为一条；窗口函数是每条记录都会执行，有几条记录执行完还是几条。
  • 聚合函数也可用于窗口函数。

窗口函数按照功能可以分为序号函数，分布函数，前后函数，头尾函数，其他函数，如下图：

3.2、窗口函数基本用法

over关键字用来指定函数执行的窗口范围，若后面括号中什么都不写，则意味着窗口包含满足where条件的所有行，窗口函数基于所有行进行计算；如果不为空则支持如下4种语法来设置窗口：

  1. window_name：给窗口指定一个别名。如果SQL种涉及的窗口较多，采用别名可以看起来更清晰易读。
  2. partition by：窗口按照哪个字段进行分组，窗口函数在不同的分组上分别执行。
  3. order by：按照哪些字段进行排序，窗口将按照排序后的记录顺序进行编号。
  4. frame：frame是当前分区的一个子集，子句用来定义子集的规则，通常用来作为滑动窗口使用。
