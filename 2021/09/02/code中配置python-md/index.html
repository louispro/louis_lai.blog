在VScode中配置python的运行环境


一、安装VS code

下载VS code

安装直接按照程序提示安装即可

二、安装python

下载 python3.8.5

2.1、安装步骤

  1. 选择自定义安装

  2. 勾选全部点击next

  3. 如下图勾选，然后点击install

  4. 验证python是否安装成功

    打开命令提示符窗口输入python，如下图则说明安装成功

三、在VS code中配置python

3.1、VS code设置为中文

  1. 按下快捷键Ctrl+shift+p打开搜索框

  2. 在搜索框中输入configure language,结果如下图

  3. 添加语言

  4. 点击install然后重启VS code

3.2、安装第三方包为配置做准备

pip flake8，flake会检查编写代码的不规范的地方和语法错误

pip yapf，yfpf是一个代码格式化工具，可以一键美化代码

查看是否安装成功：pip list

3.3、安装python插件(必装)

  1. 在VS code下按下快捷键ctrl+shift+x，进入插件管理界面

  2. 在搜索栏输入python

  3. 选择插件点击安装

    ![python plugins](./code中配置python-md/python plugins.png)

3.4、安装辅助插件(可选)

  1. vscode-icon

    区分文件类型的图标

  2. Chinese

    汉化插件

  3. arepl

    随着你输入代码，会同时显示代码执行结果

3.5、创建项目

  1. 创建虚拟环境

    在windows终端下输入py -3 -m venv env创建虚拟环境，生成如下目录：

  2. 将项目解释器切换成刚创建的虚拟环境

    按下快捷键ctrl+shift+p打开命令框输入python:select配置解释器

  3. 配置好虚拟环境之后运行测试代码

  4. 解决VSCODE”因为在此系统上禁止运行脚本”报错

    配置好虚拟环境之后需要通过Activate.ps1激活环境之后才能运行python脚本。但有可能报”因为在此系统上禁止运行脚本”的错误，这是由于PowerShell执行策略的问题。

    1. 以管理员身份运行vscode
    2. 在terminal中输入：get-ExecutionPolicy显示Restricted，表示状态是禁止的
    3. 在terminal中输入：set-ExecutionPolicy
    4. 这时再执行get-ExecutionPolicy，就显示RemoteSigned;
